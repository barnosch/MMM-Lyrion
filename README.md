Clone to modules folder<br>
https://github.com/barnosch/MMM-Lyrion.git<br>

cd MMM-Lyrion<br>
npm install  

Add to your config.js\
{\
    disabled: false,\
    module: "MMM-Lyrion",\
    position: "bottom_right",\ 
    config: {\
        updateInterval: 30000,                // optional, Standard: 60000ms\
        lmsServer: "http://10.30.10.11:9000"  // IP:9000\
      }\
    },\
